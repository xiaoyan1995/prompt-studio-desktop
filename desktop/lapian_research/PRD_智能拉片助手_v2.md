# Prompt Studio — 智能拉片助手 PRD v2

> **版本**: v2.0（基于 FM80cine 逆向代码重写）  
> **日期**: 2026-05-21  
> **参考竞品**: FM80cine 拉片助手  
> **数据来源**: QlHmieW4.js（API 端点）、uHOAdXJl.js（Shot 组件）、Dl12JbVT.js（详情页）、gQCvDTnZ.js（搜索维度）  
> **原则**: 功能 1:1 对齐 FM80cine，体验做到本地极致

---

## 一、产品定位

在 Prompt Studio 左侧导航栏新增「🎬 智能拉片」入口，作为与「📁 所有项目」并列的**独立子系统**。用户拖入本地视频 → 自动分镜 → AI 逐镜头生成「镜头摘要 + 视听描述 + AIGC提示词」→ 输出可直接用于 Sora/Kling/FLUX 的影视级提示词。

### 对比优势

| | FM80cine (Web) | Prompt Studio (本地) |
|---|---|---|
| 视频限制 | 300MB, 需上传 | **无限制**, 本地读取 |
| 分镜速度 | 上传+排队, 分钟级 | **本地 FFmpeg, 秒级** |
| AI 费用 | 平台限次 | **自己的 Key, 无限** |
| 隐私 | 视频上传三方 | **纯本地, 仅图片发AI** |
| 资产联动 | 无 | **一键存入项目资产库** |

#### 3.6.4 右侧 — AI 分析内容（核心）

完全对标 FM80cine 截图中的分析结构，由 AI 生成 Markdown 格式文本：

---

**镜头摘要** `shotSummary`

> 一段叙事性的总结文字，200-400字，描述该镜头在影片中的意义、表达手法和情感传递。

---

**视听描述**（结构化分析，对标截图中的 4 大板块）

**1. 镜头信息**
- **景别** `shotScale`：极特写(ECU)/特写(CU)/中近景(MCU)/中景(MS)/中远景(MLS)/远景(LS)/全景(WS)/大全景(EWS) — 附详细说明
- **角度** `cameraAngle`：平视/仰拍/俯拍/鸟瞰/荷兰角 — 附详细说明
- **运镜** `cameraMovement`：固定/推/拉/横摇/纵摇/跟拍/手持/航拍/变焦 — 附详细说明

**2. 画面构成** `composition`
- **场景与层次**：拍摄空间描述（室内/室外、前景/中景/背景层次、空间纵深感）
- **构图规则**：采用什么构图法则（对称/三分法/引导线/框架/黄金分割等）

**3. 色彩与光影**
- **色调倾向** `colorAnalysis`：主色调、饱和度、色温风格
- **光线** `lighting`：光源方向、软硬度、色温、光影对比

**4. 声音推断**
- 推断背景音乐、环境音效、对白风格（基于画面推断，因为我们处理的是无声关键帧）

### 3.8 AI 多模态分析引擎

#### 3.8.1 分析触发

分镜完成后，系统提示「🤖 AI 自动分析所有镜头」，用户确认后开始。

对标 FM80cine 的 `content-analysis` 状态机：
- `pending` → `analyzing` → `completed` / `failed`

#### 3.8.2 每个镜头的分析流程

1. 读取该镜头的 **Middle Frame**（或多帧）转 Base64
2. 调用多模态 LLM（用户配置的 GPT-4o / Gemini Pro / 其他）
3. System Prompt 生成结构化 JSON
4. 解析后写入本地，前端实时更新

#### 3.8.3 AI Agent System Prompt

```
你是一个电影学院级的视听语言分析专家。你精通景别、运镜、光影设计和叙事手法。

请分析下面的电影/视频关键帧图片，并严格以如下 JSON 格式返回：

{
  "shotSummary": "200-400字的镜头叙事摘要，描述该镜头的视觉表达、在影片中的功能和情感传递",
  
  "shotScale": "景别判断 + 50字说明（如：全景(大全景)，拍摄人物与地标标识的正面关系）",
  "cameraAngle": "角度判断 + 50字说明（如：平视观察角度，贴合人眼日常观察视线）",
  "cameraMovement": "运镜判断 + 50字说明（如：固定机位长镜头，镜头保持完全静止不动）",
  
  "composition": {
    "sceneAndLayers": "场景与层次描述，80-150字，包括拍摄空间、前中后景层次、空间纵深",
    "compositionalRules": "构图规则分析，80-150字，使用的构图法则及其效果"
  },
  
  "colorAnalysis": "色调倾向分析，50-100字，包括主色调、饱和度、色温风格、情绪表达",
  "lighting": "光线分析，50-100字，包括光源方向、软硬度、色温、光影对比效果",
  
  "soundInference": "声音推断，50-100字，基于画面推断可能的背景音乐风格、环境音效、对白节奏",
  
  "aigcPrompt": "英文 AIGC 提示词，80-150 words，可直接用于 Sora/Midjourney/FLUX，包含景别、运镜、光影、主体、氛围、色调、质感"
}
```

#### 3.8.4 分析进度

- 每个镜头卡片显示状态图标：
  - ⚪ `pending` 未分析
  - 🔄 `analyzing` 分析中
  - ✅ `completed` 已完成
  - ❌ `failed` 失败（可重试）
- 顶部横幅：「AI 正在分析... (5/15 个镜头)」
- 支持单镜头重试：`retry-content-analysis`

#### 3.9.3 删除镜头

#### 3.9.4 添加镜头

对标 FM80cine `ADD_SHOT`：标记起止时间 → 创建新镜头

#### 3.9.5 从关键帧创建镜头

对标 FM80cine `CREATE_SHOT_FROM_IMAGE`：从已有图片创建一个新镜头

#### 3.9.6 镜头内图片管理

对标 FM80cine 的图片操作：
- **上传图片** `UPLOAD_IMAGES`：为镜头手动添加关键帧

- **移动图片** `IMAGE_MOVE` / `IMAGE_BATCH_MOVE`：将图片从一个镜头移到另一个
- **设置主图** `SHOT_PRIMARY_IMAGE`：指定某张为镜头封面

#### 3.9.8 编辑镜头描述

对标 FM80cine `updateShotDescription`：在分镜模式下直接编辑镜头的描述文字
