  Prompt Studio Desktop  v1.0.3  使用说明
═══════════════════════════════════════════════════

【目录结构】
  解压后应保持如下结构，请勿改变文件夹相对位置：

  📁 任意位置/
  ├── 📁 extension/            ← 浏览器插件目录
  ├── 📁 Prompt Studio Desktop/ ← 主程序目录
  │   ├── Prompt Studio Desktop.exe  ← 双击运行
  │   └── ...
  └── 📄 使用说明.txt

  ⚠️ extension 文件夹必须与「Prompt Studio Desktop」
     文件夹处于同一目录，否则插件联动功能无法使用。

──────────────────────────────────────────────────

【启动方式】
  双击 Prompt Studio Desktop / Prompt Studio Desktop.exe
  首次启动约需 3~5 秒加载本地服务。

【数据存储】
  用户数据自动保存在：
  Prompt Studio Desktop/studio-data/
  （与主程序 .exe 同目录）

  ✅ 迁移数据只需复制整个 studio-data 文件夹即可。

──────────────────────────────────────────────────

【浏览器插件安装（Chrome / Edge）】
  1. 打开浏览器，访问 chrome://extensions/
  2. 开启右上角「开发者模式」
  3. 点击「加载已解压的扩展程序」
  4. 选择 extension/ 文件夹
  5. 插件安装完成，图标出现在浏览器工具栏

  插件功能：在任意网页一键将提示词/图片发送到 Prompt Studio。

──────────────────────────────────────────────────

【常见问题】
  Q: 启动后白屏 / 无法加载？
  A: 稍等 5 秒让本地服务启动完成，或重启软件。

  Q: 提示「本地服务启动失败」？
  A: 确认 Prompt Studio Desktop/resources/server/
     目录下存在 prompt-studio-server.exe。

  Q: 插件无法连接软件？
  A: 确认软件正在运行，并检查 extension/ 目录
     位置是否与「Prompt Studio Desktop」文件夹同级。
