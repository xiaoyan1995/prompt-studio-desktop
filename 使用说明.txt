  Prompt Studio Desktop  v1.1.0  使用说明
═══════════════════════════════════════════════════

【目录结构】
  解压后目录结构如下：

  📁 Prompt Studio Desktop-1.1.0-win/
  ├── � Prompt Studio Desktop.exe  ← 双击运行
  ├── � extension/                 ← 浏览器插件
  ├── 📁 skills/                    ← Agent Skills
  │   └── prompt-studio/
  │       ├── SKILL.md
  │       └── REFERENCE.md
  └── ...（其他程序文件）

  ⚠️ 请勿改变文件夹的相对位置。

──────────────────────────────────────────────────

【启动方式】
  双击 Prompt Studio Desktop / Prompt Studio Desktop.exe
  首次启动约需 3~5 秒加载本地服务。

【数据存储】
  用户数据自动保存在：
  Prompt Studio Desktop/studio-data/
  （与主程序 .exe 同目录）

  ✅ 迁移数据只需复制整个 studio-data 文件夹即可。

──────────────────────────────────────────────────

【浏览器插件安装（Chrome / Edge）】
  1. 打开浏览器，访问 chrome://extensions/
  2. 开启右上角「开发者模式」
  3. 点击「加载已解压的扩展程序」
  4. 选择 extension/ 文件夹
  5. 插件安装完成，图标出现在浏览器工具栏

  插件功能：在任意网页一键将提示词/图片发送到 Prompt Studio。

──────────────────────────────────────────────────

【常见问题】
  Q: 启动后白屏 / 无法加载？
  A: 稍等 5 秒让本地服务启动完成，或重启软件。

  Q: 提示「本地服务启动失败」？
  A: 确认 Prompt Studio Desktop/resources/server/
     目录下存在 prompt-studio-server.exe。

  Q: 插件无法连接软件？
  A: 确认软件正在运行，并检查 extension/ 目录
     位置是否与「Prompt Studio Desktop」文件夹同级。

──────────────────────────────────────────────────

【音效库使用】
  进入项目 → 音效库，链接本地音效文件夹后：
  · 点击 ▶ 播放音效，点击卡片其他区域选中文件
  · Ctrl+点击 追加多选；空白处拖动鼠标 框选多个
  · 拖拽选中的卡片到 DAW / 文件管理器（原生拖拽）
  · 标题栏 📌 按钮可将窗口置顶，方便边看边拖
  · 「翻译音效名」按钮调用 LLM 自动翻译为中文
    （需在「设置 → 文本 LLM」填写 API 配置）

【Agent Skills 使用】
  skills/prompt-studio/ 目录是给 AI agent 用的技能文件。
  将 SKILL.md 的内容加入 agent 的 system prompt，
  agent 即可通过 HTTP API 读写 Prompt Studio 的提示词、
  图片、视频等内容，无需安装任何额外工具。

  本地服务地址：http://localhost:8767
